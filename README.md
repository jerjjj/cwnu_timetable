# CWNU Timetable 

​                                   专为西华师范大学学子设计的课表软件

## 下载链接

安卓用户请前往gitee下载最新稳定版：[xifankebiao: 稀饭课表软件更新服务](https://gitee.com/jerjjj_admin/xifankebiao)

想要提前尝鲜新功能可以前往***github action***下载测试版

> [!WARNING]
>
> 注意，测试版不建议日常使用

对于苹果用户，可以在***github action***下载ipa安装包，但是需要进行自签

## 软件功能

- [x] 课表查看
- [x] 课表自动更新
- [x] 软件内更新
- [x] 桌面小组件

- [ ] 成绩查看
- [ ] 考试信息查询

- [ ] 设定自定义事项
